Render the shadow map for the light.

This method will render the scene from the light's perspective to create a depth map, which is used for shadow mapping.

Parameters
shaderusage of the shader class is needed to easily bind the light to fragment and vertex, depending on the graphic api.
scenethe scene where the light is rendered, used to access entities and render them in the shadow map.

Implements LightData.
