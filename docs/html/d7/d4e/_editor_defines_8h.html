Detailed Description

Defines macros and helper functions used specifically in editor or game builds.

This header provides a mechanism for conditionally including/excluding code based on whether the engine is built in editor or runtime (game) mode.

Use the following macros:

  • EDITOR_ONLY(code): Includes code only in editor builds.
  • IN_GAME_ONLY(code): Includes code only in game builds.
  • EDITOR_LOG(msg): Logs editor messages with timestamps to std::cout.
  • EDITOR_ERROR(msg): Logs error messages with timestamps to std::cerr.
Note
Define ENGINE_EDITOR at compile time to enable editor features.
