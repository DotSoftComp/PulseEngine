◆ GetScriptFromCallName()

IScript * ScriptsLoader::GetScriptFromCallName ( std::string callName)
static

Get the Script From Call Name object.

This function will retrieve the script from the dll and create the IScript*.

If no dll loaded, it will try to load it. No crash if the dll is not loaded. It will just return a nullptr.

Parameters
callNameName of the function to call in the dll.
Note
The function should be exported from the dll with the name "PulseScript"
Returns
IScript* the actual script. it will be a child that inherit from this interface but its not important, it has the Update and Start functions.
