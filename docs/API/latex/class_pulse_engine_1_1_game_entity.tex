Instantiate an entity from a path to a file, position, rotation and scale. 

The instantiation will directly call the Start() method from the entity, and also all start methods from the scripts attached to the entity.

If no entity could have been instantiated, it will return a nullptr. \begin{DoxyNote}{Note}
By passing an Entity\texorpdfstring{$\ast$}{*} to this method, by replacing path, you can instantiate an already existing entity in the world, saving ressources from loading it again.
\end{DoxyNote}
