static

Instantiate an entity from a path to a file, position, rotation and scale.

The instantiation will directly call the Start() method from the entity, and also all start methods from the scripts attached to the entity.

If no entity could have been instantiated, it will return a nullptr.

Note
By passing an Entity* to this method, by replacing path, you can instantiate an already existing entity in the world, saving ressources from loading it again.
Parameters
pathThe path of the .pentity file. With this method you can only instantiate entities that are loaded in the engine.
positionPosition where the entity will be instantiated.
rotationRotation of the entity in degrees (pitch, yaw, roll).
scaleScale of the entity, we recommend to use the same scalar on each axis to keep the entity uniform.
Returns
Entity* The pointer to the entity instantiate in the world. You can directly modify it after the instantiation.
